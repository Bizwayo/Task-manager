TASK MANAGER
====================

This is a Task manager API developed with Nodejs using MongoDB as the database to store information.

RUN
=====
- Make sure you have npm installed on your local
- Clone the repo
- Open cmd and go to the folder
- On cmd, enter 'npm init' to install all dependencies
- To start the server enter 'npm start', default port will be 3000
- In your web browser, go to 'http://localhost:3000/'
